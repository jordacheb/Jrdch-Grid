Minimal Responsive CSS Grid

Usage guide

Layout

There is an ID #jrdch that is used to encapsulate the code in case you put it inner a existing estructure in order to avoid conflict.

Control the page width by changing #jrdch-wrapper width in CSS.

<body>
  <div id="jrdch">
    <div id="jrdch-wrapper">
      ...here goes the grid...
    </div>
  </div>
</body>

Grid

The basic structure uses a div .g-block-group to define a row where you can set a .g-block div with 100% width or more divs like .col-20 where the number 20 stands for the % of the column width in CSS.

<div class="g-block-group">
  <div class="g-block col-20">
    ...content in a 20% width column...
  </div>
  <div class="g-block col-80">
    ...content in a 80% width column...
  </div>
</div>

Helper Classes

No padding left / No padding right

Every column has a 20px padding in it's sides. If you need to remove it simply use one of these classes:

.npl {
  padding-left: 0;
}
.npr {
  padding-right: 0;
}
